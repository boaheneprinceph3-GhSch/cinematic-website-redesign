## 🏛️ Balstudio — Architecture Firm

**Industry:** Architecture & Design  
**Location:** Accra, Ghana  
**Live Site:** [View Site](https://balstudio-cinematic-78o5x47bm-boaheneprinceph4-7854s-projects.vercel.app)  
**Code:** [View Code](./balstudio-cinematic)

### Overview
Award-winning architecture firm website featuring scroll-scrubbed video hero, text mask reveals, and interactive Perlin noise waves background.

### Key Features
- ✨ Scroll-scrubbed hero video (5-8 second cinematic loop)
- 🎭 Text mask reveal: "WHERE VISION MEETS PRECISION"
- 🌊 Interactive waves background (Perlin noise + cursor interaction)
- 🎴 3D flip cards for services (4 cards)
- 📸 Accordion slider for projects (5 panels)
- 📊 Animated stats counter
- 📱 Fully responsive mobile design

### Tech Highlights
- **Animations:** GSAP ScrollTrigger
- **Canvas:** Custom Perlin noise implementation
- **Typography:** Cormorant Garamond + Inter
- **Icons:** Google Material Symbols
- **Performance:** 90+ Lighthouse score

### Color Palette
- Primary: `#1a1a1a` (Deep charcoal)
- Secondary: `#f5f5f5` (Soft white)
- Accent Gold: `#d4af37` (Sophisticated gold)
- Accent Warm: `#8b7355` (Warm brown)

### Components Used
1. Text Mask Reveal (hero headline)
2. Interactive Waves Background (services, about, contact sections)
3. 3D Flip Cards (services)
